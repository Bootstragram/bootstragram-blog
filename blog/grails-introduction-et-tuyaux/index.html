Grails : introduction et tuyaux

November 18, 2009 •

    Grails est un framework de développement qui permet de créer des applications web très rapidement. Ce produit, construit autour de Java et de Groovy est particulièrement performant pour développer des applications qui sont très proches d’une simple manipulation d’un modèle de données.

    Rappels des commandes de base

    • grails help

      Obtenir de l’aide sur Grails

    • grails run-app

      Fait tourner l’application Grails dans Jetty

    Plugins indispensables

    • acegi

      Authentification et contrôles d’accès

    • image-tools

      Chargement, sauvegarde et manipulation d’images (crop, mask, création de thumbnail)
